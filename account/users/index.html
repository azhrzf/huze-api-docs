/users

  • GET /users/{id}

GET /users/{id}

Parameter: id

Example: users/u1sansajf

Body:

{
    "email": "uremail@gmail.com",
    "password": "urpassword",
    "fullname": "Ur Full Name"
}

Response:

{
    "status": "success",
    "data": {
        "user": {
            "id": "urUserId",
            "email": "uremail@gmail.com",
            "fullname": "Ur Full Name"
        }
    }
}
